# 🧮 Tax Calculator

A lightweight, browser-based tax calculator built with vanilla HTML, CSS, and JavaScript — no frameworks or dependencies required.

## 🚀 Live Demo

> Deployed via GitHub Pages: https://mariodrs.github.io/taxcalculator

## 📋 Features

- Calculate tax based on income input
- Clean, responsive UI
- Instant results with no page reloads
- Runs entirely in the browser — no backend needed

## 🗂 Project Structure

```
taxcalculator/
├── index.html        # Main HTML structure
├── style.css         # Styling and layout
├── script.js         # Tax calculation logic
└── .github/
    └── workflows/    # CI/CD via GitHub Actions
```

## 🛠 Getting Started

No installation or build step needed. Just clone and open in a browser:

```bash
git clone https://github.com/mariodrs/taxcalculator.git
cd taxcalculator
open index.html
```

Or simply visit the link above.

## ⚙️ GitHub Actions

This project uses a GitHub Actions workflow for automated deployment (e.g. to GitHub Pages) on every push to `main`.
